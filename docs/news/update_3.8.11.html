Оновлення корпоративного сервісу «Арт-Офіс» до версії 3.8.11

2025-10-17 /
Сервіс «Арт‑Офіс»

Шановні користувачі!

Повідомляємо, що проведено оновлення сервісу обміну електронними документами «Арт‑Офіс»:

Зміни у версії 3.8.11:

  • Робота з КЕП:
    • Оновлено кореневі сертифікати КНЕДП "УСС" та додано новий кореневий сертифікат OCSP-сервісу КНЕДП АТ "Банк-Альянс"
    • Оновлено кореневі сертифікати ДПС "ОТРИМАНО" та "ОТРИМАНО ЕК" для подання звітності ФОП
    • Оновлено кореневі сертифікати КНЕДП “ТАКСОМБАНК"
  • Безпека:
    • Оновлено WebCryptoService до версії 1.1.16.4465
  • Інший функціонал:
    • Доопрацьовано умову транслітерації українського алфавіту латиницею для літери "Ч"
    • Доопрацьовано шаблон первинного документа PD1C8210 - Акт звіряння
    • Доопрацьовано верстку футера головної сторінки сайту
    • Доопрацьовано сторінку "Створення/Редагування/Перегляду маршруту обробки документа"
    • Доопрацьовано функціонал 'Історія дій з документом', додано інформацію про нотифікацію
    • Доопрацьовано перевірку кількості символів для параметру "Назва організації" / "ПІБ користувача"
    • Додано відступ між іконкою перегляду та назвою контрагента на сторінках перегляду документа для ролі "Глобальний адміністратор"
    • Вилучено обмеження сортування за періодом створення для документів організації на сторінці "Документи на підпис"
  • Міграція на новий шаблон сайту з використання React/TypeScript:
    • Заблоковано можливість fallback для ролей "Глобальний адміністратор" та "Адміністратор ВЕД"
    • Реалізовано коректну обробку посилань з листів в рамках міграції на новий шаблон сайту з використання React/TypeScript
    • Роль "Оператор ВЕД":
      • Актуалізовано іконки для розділів у відповідності до попереднього шаблону сайту
      • Реалізовано функціонал для сторінки "Пошук"
      • Реалізовано функціонал для контролів "Tag Cloud" та "Період створення документів"
      • Реалізовано функціонал для фільтрації документів ВЕД
      • Доопрацьовано верстку сторінки відкриття окремого документа
      • Реалізовано функціонал для сторінки фільтрів документів ВЕД
      • Реалізовано функціонал точкового підпису файловими ключами та ключами, що зберігаються в хмарі, для модального вікна "Підписання документа" та сторінки перегляду документа
      • Реалізовано функціонал масового підпису файловими ключами та апаратними ключами (в тому числі мережного криптомодуля Gryada301 (МК)) для розділу документів
    • Роль "Оператор документів":
      • Актуалізовано іконки для розділів у відповідності до попереднього шаблону сайту
      • Реалізовано функціонал підписання та шифрування для "Запит до бюджета" файловими та апаратними ключами (в тому числі мережного криптомодуля Gryada301 (МК)), а також ключами, що зберігаються в хмарі
      • Реалізовано функціонал для фільтрації ЗЕД документів (Вихідні, Вхідні, Недавні)
      • Реалізовано функціонал для контролів "Tag Cloud" та "Період створення документів"
      • Реалізовано функціонал створення документів
      • Реалізовано функціонал для сторінок "Запити на видалення", "Пошук коментарів", "Пошук", "Запити до бюджету", "Фінмоніторинг" -> "Всі запити (ФОП)", "Документи Мій Арт-Офіс", "Документи MEDOC", "Звіти (ФОП)"
      • Реалізовано функціонал точкового підпису файловими ключами та ключами, що зберігаються в хмарі, для модального вікна перегляду документа ЗЕД
      • Реалізовано функціонал параметрів пошуку для сторінки коментарів з email
      • Реалізовано функціонал параметрів пошуку для сторінки "Видалені документи"
      • Реалізовано функціонал перегляду документа на окремій сторінці
    • Роль "Оператор ФінМон":

      • Реалізовано функціонал отримання інформації для Анкети ФінМон ФОП від ДПС по запиту, підписаного файловим та ключем, що зберігається в хмарі

      • Реалізовано функціонал для сторінок "Всі запити" та "Пошук запитів"
      • Реалізовано функціонал для сторінок заповнення\перегляду форми опитувальника та анкети ризику
    • Роль "Супервайзер":
      • Реалізовано функціонал точкового підпису файловими ключами та ключами, що зберігаються в хмарі, для модального вікна "Підписання документа" та сторінки перегляду документа
      • Реалізовано функціонал масового підпису файловими ключами та апаратними ключами (в тому числі мережного криптомодуля Gryada301 (МК)) для розділу документів
      • Реалізовано функціонал масового підпису ключами, що зберігаються в хмарі, для розділу документів
      • Реалізовано функціонал масового одночасного підпису файловими ключами підпису та печатки для розділу документів
      • Реалізовано функціонал вибору періоду відображення документів для сторінки "Організації" та фільтрації документів
      • Доопрацьовано верстку сторінки організацій для ролі "Супервайзер"
