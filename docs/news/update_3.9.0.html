Оновлення корпоративного сервісу «Арт-Офіс» до версії 3.9.0

2026-06-11 /
Сервіс «Арт‑Офіс»

Шановні користувачі!

Повідомляємо, що проведено оновлення сервісу обміну електронними документами «Арт‑Офіс»:

Зміни у версії 3.9.0:

  • Робота з КЕП:
    • Оновлено та додано кореневі сертифікати КНЕДП ДП «Дія»
    • Оновлено та додано кореневі сертифікати КНЕДП АТ «КРЕДІ АГРІКОЛЬ БАНК»
    • Додано нові сертифікати ZC NBU
    • Додано два CMP-сертифікати для КНЕДП ТОВ «Депозит Сайн»
    • Додано новий ланцюжок кореневих сертифікатів КНЕДП ТОВ «Депозит Сайн»
    • Додано новий ланцюжок кореневих сертифікатів КНЕДП Держказначейства
    • Додано новий ланцюжок кореневих сертифікатів КНЕДП АЦСК АТ КБ «ПРИВАТБАНК»
    • Оновлено кореневий OCSP-сертифікат КНЕДП ТОВ «Центр сертифікації ключів Україна»
    • Додано підтримку нового КНЕДП EDIN ID ТОВ «АТС»
    • Оновлено OCSP-/кореневі сертифікати ЦЗО
    • Додано сертифікати для підтримки алгоритму ECDSA
  • Безпека:
    • Покращено перевірку сертифіката сервера — тепер відображається детальніше повідомлення про помилку
    • Змінено алгоритм хешування для ASiCManifest.xml з ГОСТу на ДСТУ, значення винесено в Config
    • Додано захист від HTML-ін'єкцій у методах створення контрагентів та користувачів
    • Оновлено JavaScript та серверну криптобібліотеку до версії 2.8.45.4515 з підтримкою алгоритму ДСТУ 7564:2014 («Купина»)
    • Оновлено криптосервіс до версії 1.1.17.4506
    • Додано ознаки forceDstuHash та suppressSignaturesParse для примусового вибору алгоритму хешування (ДСТУ замість ГОСТ) та відключення парсингу підписів для несумісних форматів (PAdES RSA, DocuSign тощо)
    • Оновлено ІІТ криптобібліотеку до версії 1.3.73
    • Підключено оновлену криптобібліотеку DSTU4145-DSTU7564 SmartID
    • Додано ознаку suppressSignaturesParse для примусового відключення парсингу підписів при несумісному форматі документа (PAdES RSA тощо)
    • Заблоковано операції підписання в кабінеті користувача при вході через Active Directory
    • Оновлено вміст QR-коду документа
  • Новий UI та міграція на React/TypeScript:
