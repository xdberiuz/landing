Новий інтерфейс користувача в «Арт-Офіс»!

2025-10-17 /
Сервіс «Арт‑Офіс»

Шановні користувачі!

У версії «Арт-Офіс» 3.8.11 ми реалізували повну міграцію системи на новий шаблон сайту з використанням React/TypeScript.

Тепер оновлений інтерфейс користувача (UI) доступний для всіх ролей у системі.

Новий дизайн має сучасний вигляд, покращену швидкодію та зручнішу навігацію. Разом із цим ми продовжуємо адаптувати всі модулі та функції під нову платформу.

Звертаємо увагу, що:

  • для ролей «Глобальний адміністратор», «Локальний адміністратор» та «Адміністратор ВЕД» новий інтерфейс увімкнено обов’язково;
  • для ролей «Оператор ВЕД», «Оператор документів», «Оператор ФінМон» і «Супервайзер» поки що залишається можливість перемикання між новим і попереднім виглядом системи.

Під час роботи з оновленим інтерфейсом можуть відрізнятися окремі елементи або тимчасово з’являтися неточності в роботі сторінок. Ми поступово вдосконалюємо всі елементи UI та будемо вдячні за ваші відгуки.

Якщо маєте побажання, зауваження чи пропозиції щодо нового інтерфейсу або помітите помилки в роботі, будь ласка, повідомте нашу команду підтримки — ваші відгуки допомагають нам зробити систему ще кращою!

З повагою,
Команда розробників «Арт‑Офіс»
